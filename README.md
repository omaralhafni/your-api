## 🏠 **Running the project locally** <span id='install'></span>

1. Clone the repo
   ```sh
   git clone https://github.com/omaralhafni/your-api.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your API in `.env`
   ```js
   const REACT_APP_URL = 'ENTER YOUR API';
   ```
